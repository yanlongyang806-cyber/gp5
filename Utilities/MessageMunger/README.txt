MessageMunger is a tool to convert old-style .ms files to new-style .ms files and back.
It is currently used to handle TexWords translation issues.

Usage:
	MessageMunger old2new ...
	MessageMunger new2old ...

Building:
	Install Python and Py2exe:
		http://www.python.org
		http://www.py2exe.org/

	Run: python setup.py py2exe

	Copy the "dist" directory's contents to c:\Cryptic\tools\bin.
